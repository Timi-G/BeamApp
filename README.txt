
The app UI is built using kivy, a Cross-PlatForm Python Framework that implements NUI (Natural Use Interface)


To test app and access data entry screen:
1) Start app from the TheMainApp.py file
2) Click 'login' button
3) Click 'DataBook' icon
4) Populate data entry screen as desired

You can try out the custom reminder feature by selecting 'Remind me' radio button and choosing 'Custom' reminder option.
